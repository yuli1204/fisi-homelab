# FISI Homelab – Uptime Kuma (Docker)

Dies ist ein kleines Lernprojekt für meine Bewerbung als  
**Fachinformatiker/in – Systemintegration**.

## Projektbeschreibung
In diesem Projekt habe ich Docker Desktop unter Windows installiert  
und mit Docker Compose einen Monitoring-Dienst eingerichtet.

Der Dienst heißt **Uptime Kuma** und läuft in einem Container.  
Er ist über den Webbrowser erreichbar.

## Verwendete Technologien
- Windows 10/11
- Docker Desktop
- Docker Compose
- Uptime Kuma

## Projektinhalt
- docker-compose.yml zum Starten des Containers
- Web-Dashboard über http://localhost:3001
- Persistente Daten mit Docker Volumes

## Was ich gelernt habe
- Installation und Konfiguration von Docker
- Arbeiten mit Docker Compose
- Starten und Verwalten von Containern
- Grundlegendes Verständnis von Ports und Volumes
- Nutzung von Dokumentation und Fehlersuche

## Ziel
Dieses Projekt dient als praktische Übung  
und als Nachweis meines Interesses an Systemintegration.
